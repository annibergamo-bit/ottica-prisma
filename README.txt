# Ottica Prisma - sito pronto per Vercel

## Come pubblicarlo su Vercel
1. Vai su https://vercel.com
2. Accedi
3. Clicca Add New -> Project
4. Clicca Upload
5. Trascina dentro il contenuto di questa cartella oppure carica lo ZIP
6. Clicca Deploy

## File importanti
- index.html
- styles.css
- assets/logo.png
- assets/qr-recensioni.png

## Dopo la pubblicazione
- collega il dominio Aruba nelle impostazioni Domains di Vercel
- su Aruba aggiorna i DNS come indicato da Vercel

## Nota
Nel form contatti devi sostituire `your-form-id` in index.html con il tuo vero ID Formspree.
